# Infinite Mosaic (인피니트 모자이크) — 심화 기획문서

---

## 1. 게임 비전

### 한줄 소개 (엘리베이터 피치)
**"퍼즐을 풀면 세상에 단 하나뿐인 예술 작품이 탄생하는, 무한 생성형 모자이크 퍼즐 게임"**

### 게임의 핵심 감정 (플레이어가 느껴야 할 것)
- **창작의 뿌듯함**: 내가 조립한 퍼즐이 아름다운 아트워크가 되는 순간의 성취감
- **명상적 몰입**: 조각을 하나씩 맞춰가는 과정에서 느끼는 평온함과 집중
- **소유의 만족감**: "이건 내가 만든 작품"이라는 애착, 갤러리가 채워지는 수집의 기쁨
- **미적 자부심**: 완성된 작품을 공유했을 때 "이거 예쁘다"는 반응을 받는 쾌감

### 디자인 철학 (왜 이 게임이 존재하는가)
기존 퍼즐 게임은 "정해진 이미지를 복원"하는 수동적 행위다. Infinite Mosaic은 "퍼즐을 풀면 매번 새로운 예술이 탄생"하는 능동적 창작 행위로 퍼즐의 패러다임을 전환한다. 프로시저럴 생성 알고리즘이 보장하는 무한한 콘텐츠와, 색채 이론 기반의 자동 팔레트 시스템이 보장하는 미적 품질이 만나, "누구나 예술가가 될 수 있는" 경험을 제공한다. 서버 없이, 업데이트 없이, 영원히 새로운 퍼즐을 풀 수 있는 지속 가능한 구조가 이 게임의 존재 이유다.

### 타겟 유저 프로필

| 구분 | 페르소나 | 핵심 욕구 | 기대 만족도 |
|------|----------|-----------|-------------|
| **1차 타겟** | P2 이서연 (26세, 직장인 여성) | 출퇴근길 힐링, 예쁜 결과물 인스타 공유 | 매우 높음 (47/50) |
| **1차 타겟** | P6 한소율 (34세, 프리랜서 여성) | 아트워크 감성, 독특한 비주얼, 창작 커뮤니티 | 매우 높음 (47/50) |
| **2차 타겟** | P4 정미경 (43세, 주부) | 쉬운 조작, 명상적 플레이, 가족과 함께 | 높음 (39/50) |
| **2차 타겟** | P3 박준혁 (32세, 하드코어 게이머) | 미적 점수 최적화, 제한 피스 전략, 분석 깊이 | 보통 (33/50) |
| **2차 타겟** | P1 김하율 (18세, 고등학생) | 스피드 모자이크 아케이드, 숏폼 콘텐츠 | 보통 (31/50) |
| **2차 타겟** | P5 최동현 (22세, 대학생 남성) | 레이스 모드 친구 대결, 리더보드 경쟁 | 보통 (29/50) |
| **비타겟** | FPS/MOBA 전용 하드코어 경쟁 유저 | 이 게임의 "퍼즐" 장르 자체가 핵심 욕구와 불일치 | - |

## 2. 코어 메커니즘 상세

### 핵심 동사 (플레이어가 하는 행동)
1. **배치한다 (Place)** — 흩어진 조각을 올바른 위치로 드래그하여 놓는다
2. **회전한다 (Rotate)** — 조각을 탭하여 올바른 각도로 돌린다
3. **감상한다 (Appreciate)** — 완성된 아트워크를 감상하고 갤러리에 저장한다
4. **공유한다 (Share)** — 완성 작품/결과 카드/타임랩스를 SNS에 공유한다
5. **경쟁한다 (Compete)** — 스피드 모자이크/대결/토너먼트에서 점수를 겨룬다

### 조작 방식 (터치 제스처별 매핑)

| 제스처 | 동작 | 상세 |
|--------|------|------|
| **드래그** | 조각 이동 | 조각을 손가락으로 끌어 원하는 위치로 이동. 올바른 위치 근접 시 자석 스냅 |
| **탭** | 조각 회전 | 조각을 탭하면 90도 시계방향 회전. 더블탭 시 반시계방향 |
| **핀치 (두 손가락)** | 화면 확대/축소 | 100조각 이상 고난이도에서 세밀한 배치를 위한 줌 |
| **두 손가락 팬** | 화면 이동 | 확대 상태에서 작업 영역 이동 |
| **롱프레스** | 조각 정보 | 해당 조각의 힌트(올바른 위치 섬광) 표시 |
| **셰이크** | 힌트 사용 | 기기 흔들기로 무작위 조각 1개의 올바른 위치 표시 (보석 또는 광고 시청 필요) |

- **적응형 난이도**: 최근 5개 퍼즐의 완성 시간/힌트 사용 횟수를 분석하여 자동 추천 난이도 조절
- **자유 선택**: 적응형 추천과 별개로 유저가 직접 난이도 선택 가능
- **스피드 모자이크 전용**: 9~16조각 고정, 난이도는 제한 시간과 콤보 속도로 조절

## 3. 게임 모드별 상세

### 모드 계층 구조

```
[핵심 모드]
├── 감성 모드 (기본 퍼즐) ★ — 게임의 정체성
│
[서브 모드]
├── 스피드 모자이크 — 아케이드형 즉각 재미
├── 1v1 대결 — 비동기 점수 대결
├── 제한 피스 챌린지 — 전략적 최적화
├── 주간 토너먼트 — 주기적 경쟁
├── 콜라보 퍼즐 — 2인 협동 창작
│
[보너스 모드]
├── 일일 퍼즐 — 매일 동일 시드
└── 큐레이션 갤러리 — 감상 & 커뮤니티
```

### 감성 모드 (핵심 모드)
- **목적**: 명상적 퍼즐 경험, 아름다운 아트워크 창작
- **규칙**: 시간 제한 없음, 실패 없음. 자유롭게 조각을 배치하여 퍼즐 완성
- **승리 조건**: 모든 조각을 올바른 위치에 배치하면 완성
- **타겟 페르소나**: P2 이서연, P6 한소율, P4 정미경

### 스피드 모자이크 (서브 모드)
- **목적**: 즉각적 재미, 아케이드 경쟁, 숏폼 콘텐츠 소재
- **규칙**: 60초/90초/120초 제한 시간 내 미니 퍼즐(9~16조각) 연속 클리어. 클리어마다 콤보 배수 증가(x1.0→x1.2→x1.5→x2.0...). 3초 이내 미스 시 콤보 리셋
- **승리 조건**: 제한 시간 내 최대 클리어 수 + 콤보 점수 합산
- **레이스 모드**: 친구와 동일 시드로 동시 도전, 실시간 진행 비교
- **타겟 페르소나**: P1 김하율, P5 최동현

### 1v1 대결 (서브 모드)
- **목적**: 친구 간 비동기 경쟁
- **규칙**: 동일 퍼즐 시드를 공유하여 각자 풀기. 완성 시간 + 이동 횟수 + 힌트 미사용 보너스를 합산한 "모자이크 스코어"
- **승리 조건**: 모자이크 스코어 비교, 높은 쪽이 승리
- **타겟 페르소나**: P1 김하율, P5 최동현, P3 박준혁

### 제한 피스 챌린지 (서브 모드)
- **목적**: 전략적 깊이, 미적 점수 최적화
- **규칙**: 제한된 수의 조각(전체의 60~80%)만 사용 가능. 남은 조각을 어디에 배치할지 선택하여 미적 점수(대칭성/색상 조화도/패턴 완성도) 극대화
- **승리 조건**: 미적 점수 합산 (레이더 차트 3요소)
- **타겟 페르소나**: P3 박준혁, P5 최동현

### 주간 토너먼트 (서브 모드)
- **목적**: 주기적 접속 동기, 커뮤니티 경쟁
- **규칙**: 월~금 매일 1개 퍼즐 출제(날짜 시드), 5일간 총합 스코어로 순위 결정
- **승리 조건**: 주간 상위 10% 달성 시 한정 팔레트/프레임 보상
- **타겟 페르소나**: P3 박준혁, P5 최동현, P2 이서연

### 콜라보 퍼즐 (서브 모드)
- **목적**: 소셜 확장, 바이럴 소재, 커플/친구 함께하기
- **규칙**: 2인이 하나의 퍼즐을 반씩 나눠 조립. 각자 맡은 영역의 조각만 보임. 비동기 방식(링크 공유)
- **승리 조건**: 양쪽 모두 완성 시 합작 아트워크 생성. "○○ x △△" 크레딧 포함
- **타겟 페르소나**: P2 이서연, P4 정미경, P1 김하율

### 일일 퍼즐 (보너스 모드)
- **목적**: 매일 접속 동기, 글로벌 비교
- **규칙**: 전 세계 동일 시드 퍼즐 1개. 완성 시간으로 리더보드 순위
- **타겟 페르소나**: 전체 (가벼운 일일 루틴)

### 큐레이션 갤러리 (보너스 모드)
- **목적**: 아트 감성 극대화, 창작 동기, 커뮤니티 형성
- **규칙**: AI가 유저 작품 중 미적 점수 상위작을 자동 큐레이션. 공개 갤러리 방문/좋아요
- **타겟 페르소나**: P2 이서연, P6 한소율

## 4. 콘텐츠 시스템

### 프로시저럴 생성 알고리즘 설계

#### 패턴 생성: Wave Function Collapse (WFC) 변형
```
입력: 스타일 템플릿 (타일셋 규칙) + 난이도 파라미터 + 시드값
    │
    ▼
[1단계] 타일셋 선택
- 스타일별 기본 타일셋 로드 (이슬람 기하학, 만다라, 스테인드글라스 등)
- 각 타일에 연결 규칙(엣지 호환성) 정의
    │
    ▼
[2단계] WFC 실행
- 그리드 크기 = √(조각 수) x √(조각 수)
- 엔트로피 기반 셀 선택 → 타일 할당 → 전파 → 반복
- 실패 시 백트래킹 + 시드 변형으로 재시도 (최대 3회)
    │
    ▼
[3단계] 색상 하모니 적용
- HSL 색공간에서 조화 규칙 적용
  - 보색 배색 (Complementary)
  - 유사색 배색 (Analogous)
  - 삼각 배색 (Triadic)
  - 분할보색 (Split-Complementary)
- 시드값 기반 기본 색상(Hue) 결정 → 규칙에 따라 팔레트 자동 생성
    │
    ▼
[4단계] 품질 검증 필터
- 대칭성 점수 ≥ 임계값 (0.4)
- 색상 조화도 점수 ≥ 임계값 (0.5)
- 패턴 완성도 점수 ≥ 임계값 (0.4)
- 미달 시 시드 변형 후 재생성
    │
    ▼
[5단계] 조각 분할
- 완성 패턴을 난이도에 따른 조각 수로 분할
- 분할선은 패턴의 자연스러운 경계를 따름 (타일 경계 우선)
- 각 조각에 회전 각도 무작위 부여 (해당 난이도에서 회전 활성화 시)
    │
    ▼
출력: 조각 배열 + 정답 위치 맵 + 컬러 팔레트 + 미적 기대 점수
```

#### 미적 점수 산정 알고리즘
- **대칭성 (Symmetry)**: 수평/수직/대각선 축 기준 타일 배치 대칭도. 0.0~1.0
- **색상 조화도 (Color Harmony)**: 인접 타일 간 HSL 거리의 조화 규칙 일치도. 0.0~1.0
- **패턴 완성도 (Pattern Coherence)**: 타일 연결 규칙의 전체 충족 비율. 0.0~1.0
- **최종 미적 점수** = (대칭성 x 0.35) + (색상 조화도 x 0.35) + (패턴 완성도 x 0.30)
- 시즌별 가중치 미세 조정으로 단일 최적해 고착 방지

### 과금 동선 (구매 제안이 나오는 순간)

1. **광고 제거**: 3번째 전면 광고 노출 후 "₩4,900으로 영구 제거" 팝업
2. **스타일 팩**: 잠긴 스타일 선택 시 → "이 스타일을 해금하세요" (미리보기 제공)
3. **시즌 패스**: 시즌 리워드 트랙에서 프리미엄 보상 도달 시 → "시즌 패스로 해금" (무료 트랙 보상과 나란히 표시)
4. **보석 팩**: 힌트 사용 시 보석 부족 → "보석 충전" (보상형 광고 대안도 함께 표시)
5. **콜라보 팩**: 콜라보 퍼즐 완성 후 결과 화면 → "합작 작품에 특별 프레임을 입혀보세요"
6. **팔레트**: 큐레이션 갤러리에서 다른 유저의 한정 팔레트 작품 감상 후 → "이 팔레트를 얻으세요"

### 사운드 디자인 방향

#### BGM
- **감성 모드**: 앰비언트 피아노 + 가벼운 스트링. 4~6분 루프. BPM 60~80. 명상적이고 차분한 분위기
- **경쟁 모드**: 일렉트로닉 앰비언트. 약간의 긴장감. BPM 90~110
- **스피드 모자이크**: 업템포 일렉트로닉. BPM 120~140. 시간 압박감 연출
- **큐레이션 갤러리**: 재즈 피아노 + 더블베이스. 갤러리 감상 분위기

#### SFX
- **조각 픽업**: 부드러운 "풋" (종이 들어올리는 느낌)
- **조각 스냅**: 만족스러운 "틱" + 미세 진동 (ASMR적 쾌감)
- **조각 회전**: 가벼운 "쉭" (회전 사운드)
- **완성**: 오르골 풍 글리산도 + 잔향 긴 심벌 (성취감)
- **콤보**: 음높이가 점차 올라가는 톤 시퀀스
- **미적 점수 상승**: 하프 글리산도 (긍정적)
- **미적 점수 하락**: 저음 미세 톤 (부정적이지 않되, 변화 인지)

#### 앰비언트
- 시즌/테마에 따른 환경음 레이어 (벚꽃: 바람소리, 겨울: 벽난로 소리, 심해: 물방울 등)
- 유저 토글로 ON/OFF 가능

### 애니메이션 원칙

1. **자연스러운 물리감**: 조각 드래그 시 약간의 관성, 스냅 시 미세 바운스
2. **리워드 감각**: 완성 연출은 화려하되 과하지 않게. 빛 파티클이 퍼지며 조각이 하나로 합쳐지는 2초 애니메이션
3. **전환의 부드러움**: 화면 전환은 모두 300~500ms 페이드/슬라이드. 갑작스러운 전환 없음
4. **미적 점수 피드백**: 테두리 색상 변화는 0.5초 이상의 부드러운 그라데이션 트랜지션
5. **프레임 예산**: 모든 UI 애니메이션은 16ms(60fps) 이내 렌더링 보장

| **공유** | 네이티브 Share Sheet API | iOS/Android 플랫폼별 공유 통합 |
| **영상 생성** | ReplayKit (iOS) / MediaProjection (Android) | 타임랩스 자동 생성 |

### 핵심 기술 과제

| 과제 | 난이도 | 설명 | 해결 방향 |
|------|--------|------|-----------|
| WFC 알고리즘 품질 | 높음 | 생성 결과물의 미적 품질 편차 관리 | 품질 검증 필터(대칭성/색상/패턴) + 미달 시 재생성. 타일셋 규칙 사전 정교화 |
| 실시간 미적 점수 계산 | 중간 | 조각 배치 중 실시간 점수 업데이트 성능 | 변경된 조각 주변만 부분 재계산 (O(1) 근사). 전체 재계산은 배치 완료 후 |
| 100+ 조각 렌더링 성능 | 중간 | 다수 조각 동시 렌더링/드래그 시 성능 | GPU 인스턴싱, 화면 밖 조각 컬링, LOD 적용 |
| 타임랩스 영상 자동 생성 | 중간 | 플레이 과정을 15초 영상으로 압축 | 키 프레임 캡처(조각 스냅 시점) → 후처리 가속 렌더링 |
| 큐레이션 갤러리 서버 | 낮음 | 공개 갤러리/좋아요 기능의 경량 서버 | 메타데이터만 서버 저장(점수, 좋아요), 이미지는 로컬. 초기엔 Game Center 소셜 그래프 활용 |

| 리스크 | 영향도 | 확률 | 대응 방안 |
|--------|--------|------|-----------|

| 구독 과금 규제 (자동 갱신) | 낮음 | 낮음 | Apple/Google 구독 가이드라인 엄격 준수. 해지 방법 명확 안내 |

### 대응 방안 총괄 원칙
1. **핵심 정체성 수호**: 어떤 리스크 대응도 "풀면 예술이 되는 퍼즐"이라는 정체성을 훼손하지 않는 방향으로
2. **서버리스 우선**: 서버 의존성이 생기는 기능은 최소화하고, 항상 오프라인 폴백 제공
3. **단계적 확장**: 출시 시 핵심 기능만, 검증 후 서브 기능 점진적 활성화
4. **데이터 기반 의사결정**: 소프트 론칭 데이터로 시장 리스크를 사전 검증한 후 글로벌 론칭
