# Korean Word Puzzle (한끝차이) - 심화 기획문서

---

## 1. 게임 비전

### 한줄 소개 (엘리베이터 피치)
> "한글 자모를 추론하여 단어를 맞히는 매일의 습관 — 한국어의 아름다움을 게임으로 경험하는 문화적 퍼즐."

### 게임의 핵심 감정
- **지적 쾌감**: 자모 피드백을 분석해 정답을 좁혀가는 추론의 쾌감
- **성취감**: 적은 시도로 정답을 맞혔을 때의 "나 꽤 잘하잖아?" 뿌듯함
- **소속감/연대감**: 전국이 같은 문제를 풀고 결과를 공유하는 "오늘 거 풀었어?" 문화
- **미적 만족**: 한글 자모가 조합되어 글자가 완성되는 순간의 시각적 아름다움
- **습관의 안정감**: 매일 같은 시간에 퍼즐을 푸는 루틴이 주는 편안함

### 디자인 철학
한국어는 세계에서 가장 과학적이고 아름다운 문자 체계를 가지고 있으나, 한글의 구조적 특성(초성/중성/종성 조합)을 게임 메커니즘으로 활용한 사례는 거의 없다. 한끝차이는 "한글 자모 분리 피드백"이라는 세계 유일의 메커니즘을 통해 한국어 사용자에게만 가능한 독창적 퍼즐 경험을 제공한다. Wordle이 영어권에서 만든 "일일 퍼즐 공유 문화"를 한국어의 고유한 아름다움으로 재해석하여, 게임이자 문화 상품으로 자리 잡는 것이 목표이다.

### 타겟 유저 프로필

**1차 타겟: 습관형 퍼즐 유저 (전체 유저의 50%)**
- 20~50세 한국어 사용자, 여성 비율 약간 높음
- Wordle/끝말잇기/초성퀴즈 경험자
- 출퇴근, 점심시간에 3~5분 짧게 즐기는 직장인/주부
- 카카오톡/인스타로 결과를 공유하는 소셜 유저
- 대표 페르소나: P4 정미경(43세, 전업주부), P2 이서연(26세, 마케팅 담당자)

**2차 타겟: 경쟁/자극 추구 유저 (전체 유저의 25%)**
- 15~30세, 자모 테트리스 1v1 대결/리그 시즌제에 반응
- 짧은 세션의 실시간 경쟁을 즐김
- 대표 페르소나: P1 김하율(18세, 고등학생), P5 최동현(22세, 대학생)

**3차 타겟: 감성/문화 유저 (전체 유저의 15%)**
- 25~45세, 한글 아트 디렉션에 매력을 느끼는 유저
- 아름다운 스크린샷/위젯을 SNS에 공유
- 대표 페르소나: P6 한소율(34세, 일러스트레이터), P3 박준혁(32세, 개발자)

**비타겟**
- 액션/슈팅 등 빠른 반응을 원하는 하드코어 게이머
- 한국어를 전혀 모르는 비한국어 사용자 (2차 타겟인 한국어 학습자와 구분)
- 복잡한 전략 시스템이나 수백 시간의 깊이를 원하는 유저

## 2. 코어 메커니즘 상세

### 핵심 동사 (플레이어가 하는 행동)
1. **추론한다 (Deduce)**: 자모 피드백을 분석하여 다음 추측 단어를 결정
2. **입력한다 (Input)**: 한글 키보드로 단어를 타이핑
3. **공유한다 (Share)**: 이모지 그리드/통계 카드를 SNS에 공유
4. **배치한다 (Place)**: 자모 테트리스에서 떨어지는 자모를 배치하여 단어 완성
5. **대결한다 (Compete)**: 친구/리그에서 성적을 비교하며 경쟁

### 게임플레이 플로우차트

```
[앱 실행]
    │
    ├─→ [홈 화면] ← 위젯 탭으로도 진입
    │       │
    │       ├─→ [일일 퍼즐] (핵심 축 1)
    │       │       │
    │       │       ├─→ 단어 입력 → 자모별 피드백 → 추론
    │       │       │       ↑___________________________│
    │       │       │
    │       │       ├─→ 정답/실패 → 이모지 그리드 생성
    │       │       │       │
    │       │       │       └─→ [공유] → SNS/카카오톡
    │       │       │
    │       │       └─→ [연습하기] → 무제한 퍼즐 (광고/구독)
    │       │
    │       ├─→ [자모 테트리스] (핵심 축 2)
    │       │       │
    │       │       ├─→ [솔로] → 자모 배치 → 단어 완성 → 점수
    │       │       ├─→ [1v1 대결] → 매칭 → 90초×3R → 결과 카드
    │       │       └─→ [주간 토너먼트] → 시드 런 → 순위
    │       │
    │       ├─→ [대결] (핵심 축 3)
    │       │       │
    │       │       └─→ 초대 링크 생성 → 같은 단어 대결 → 비교 카드
    │       │
    │       ├─→ [보너스] → 초성 퀴즈 / 끝말잇기 퍼즐 / 끝말잇기 배틀
    │       │
    │       ├─→ [리그] → 시즌 등급 / 리더보드
    │       │
    │       └─→ [프로필] → 통계 / 도감 / 리캡 / 설정
    │
    └─→ [위젯] → 오늘 퍼즐 상태 / 스트릭 표시
```

### 조작 방식 (터치 제스처별 매핑)

| 제스처 | 일일 퍼즐 | 자모 테트리스 | 공통 |
|--------|----------|-------------|------|
| 탭 | 키보드 입력 | 자모 회전 | 버튼 선택 |
| 스와이프 좌우 | 커서 이동 | 자모 좌우 이동 | 화면 전환 |
| 스와이프 아래 | - | 자모 빠른 낙하 | 리스트 스크롤 |
| 길게 누르기 | 입력된 글자 상세 보기 | - | 도움말 팝업 |
| 두 손가락 핀치 | - | - | 글자 크기 조절 (접근성) |

### 난이도 곡선 설계

```
난이도
  ▲
  │                                    ┌── 어려움 (저빈도 어휘)
  │                              ┌─────┘
  │                        ┌─────┘
  │                  ┌─────┘  보통 (일반 어휘)
  │            ┌─────┘
  │      ┌─────┘
  │ ┌────┘  쉬움 (고빈도 일상어)
  │─┘
  └──────────────────────────────────→ 누적 플레이 일수
  Day1    Day7    Day30    Day90    Day180+

  * 일일 퍼즐 난이도는 요일별 변동:
    월(쉬움) → 화~목(보통) → 금(어려움) → 토(테마) → 일(유저 출제)
  * 자동 난이도 승급: 최근 14일 평균 시도 ≤ 3회 → 다음 난이도 제안
```

### 세션 설계

| 구분 | 최소 세션 | 최적 세션 | 최대 세션 |
|------|----------|----------|----------|
| 일일 퍼즐 | 1분 (직감 정답) | 3~5분 | 10분 (6회 풀시도) |
| 연습 모드 | 3분 (1문제) | 10~15분 (3~5문제) | 30분+ |
| 자모 테트리스 솔로 | 2분 (빠른 게임오버) | 5~8분 | 20분+ (고수) |
| 자모 테트리스 1v1 | 5분 (90초×3R) | 5분 | 8분 (지연 포함) |
| 친구 대결 | 2분 | 5분 | 10분 |

## 3. 게임 모드별 상세

### 핵심 축 1: 일일 퍼즐 (메인 경험)

**목적**: 매일 하나의 퍼즐을 풀고 결과를 공유하는 습관 형성
**규칙**:
- 5글자(기본) / 3~4글자(난이도 선택) 한국어 단어를 6회 이내에 맞힘
- 각 시도 후 자모 단위(초성/중성/종성) 개별 피드백
  - 🟩 해당 위치에 정확한 자모
  - 🟨 단어에 포함되지만 위치 다름
  - ⬜ 단어에 미포함
  - 🟪 해당 글자 내 자모 순서만 다름 (종성↔초성)
**승리 조건**: 6회 이내에 정답 맞히기
**타겟 페르소나**: P4 정미경(핵심), P2 이서연, P3 박준혁, P6 한소율 — 전 페르소나 대응

### 핵심 축 2: 자모 테트리스 (서브 게임)

**목적**: 일일 퍼즐과 다른 리듬의 "끝없는 도전" + 실시간 경쟁
**규칙**:
- 위에서 떨어지는 자모(ㄱ, ㅏ, ㄴ 등)를 배치하여 한글 단어 완성
- 완성된 단어는 사라지며 점수 획득, 긴 단어일수록 고득점
- 콤보 시스템: 연속 단어 완성 시 보너스 배율 상승
- 1v1 대결: 같은 자모 시퀀스, 90초×3라운드, 콤보 시 상대에게 방해 자모(ㅋ, ㅌ, ㅍ 등) 전송
**승리 조건**: 솔로 — 최고점 달성 / 1v1 — 3라운드 중 2라운드 이상 승리
**타겟 페르소나**: P1 김하율, P5 최동현(1v1 대결), P3 박준혁(최적화 플레이)

### 핵심 축 3: 대결 (소셜)

**목적**: 같은 단어를 놓고 친구와 직접 비교하는 소셜 경쟁
**규칙**:
- 초대 링크 생성 → 카카오톡/메신저 공유 → 같은 단어 도전
- 시도 횟수 + 소요 시간 비교
- 서버 불필요: 단어 시드를 초대 코드에 인코딩
**승리 조건**: 더 적은 시도 횟수 (동점 시 소요 시간)
**타겟 페르소나**: P4 정미경(가족 대결), P1 김하율(친구 대결), P2 이서연(동료 대결)

### 보너스 모드 (부가 콘텐츠)

**(A) 초성 퀴즈 모드**
- 목적: 어휘력 극한 테스트
- 초성만 제시, 시간 내 가능한 많은 정답 나열
- 타겟: P3 박준혁(최적화), P4 정미경(두뇌 활동)

**(B) 끝말잇기 퍼즐 모드**
- 목적: 솔로 퍼즐로서의 끝말잇기
- 한정된 글자 풀 내 최장 체인 만들기
- 타겟: P4 정미경, P6 한소율

**(C) 끝말잇기 배틀 모드**
- 목적: 실시간/비동기 끝말잇기 대전
- 타겟: P5 최동현, P1 김하율

### 모드 간 관계 (계층)

```
┌─────────────────────────────────────────────┐
│              핵심 (매일 방문 이유)              │
│  ┌─────────┐  ┌──────────┐  ┌─────────┐    │
│  │일일 퍼즐 │  │자모테트리스│  │  대결    │    │

│              보너스 (부가 가치)                  │

```

## 4. 콘텐츠 시스템

### 프로시저럴 생성 알고리즘 설계

#### 한국어 단어 DB 구축

**데이터 소스**: 표준국어대사전 기반 일반 명사/동사/형용사
**단어 풀 규모**:
- 쉬움: 일상 고빈도 단어 약 3,000개
- 보통: 일반 어휘 약 15,000개
- 어려움: 저빈도/전문 어휘 약 50,000개

**단어 속성 테이블 구조**:
```
{
  "word": "사랑해",
  "jamo": ["ㅅ","ㅏ","ㄹ","ㅏ","ㅇ","ㅎ","ㅐ"],
  "syllables": 3,
  "difficulty": 1,        // 1=쉬움, 2=보통, 3=어려움
  "frequency_rank": 42,   // 빈도 순위
  "category": ["감정","일상"],
  "season_tags": ["봄","여름","가을","겨울"],
  "is_filtered": false     // 비속어 필터 통과 여부
}
```

#### 자모 분리 알고리즘

```
한글 유니코드 범위: 0xAC00 ~ 0xD7A3
글자 = 0xAC00 + (초성 인덱스 × 21 × 28) + (중성 인덱스 × 28) + 종성 인덱스

초성 (19개): ㄱ ㄲ ㄴ ㄷ ㄸ ㄹ ㅁ ㅂ ㅃ ㅅ ㅆ ㅇ ㅈ ㅉ ㅊ ㅋ ㅌ ㅍ ㅎ
중성 (21개): ㅏ ㅐ ㅑ ㅒ ㅓ ㅔ ㅕ ㅖ ㅗ ㅘ ㅙ ㅚ ㅛ ㅜ ㅝ ㅞ ㅟ ㅠ ㅡ ㅢ ㅣ
종성 (28개): (없음) ㄱ ㄲ ㄳ ㄴ ㄵ ㄶ ㄷ ㄹ ㄺ ㄻ ㄼ ㄽ ㄾ ㄿ ㅀ ㅁ ㅂ ㅄ ㅅ ㅆ ㅇ ㅈ ㅊ ㅋ ㅌ ㅍ ㅎ

분리 함수:
  input: 글자 (char)
  code = char - 0xAC00
  초성 = code / (21 × 28)
  중성 = (code % (21 × 28)) / 28
  종성 = code % 28
  return (초성, 중성, 종성)
```

#### 일일 퍼즐 시드 생성

```
seed = parseInt(YYYYMMDD)  // 예: 20260326
rng = PRNG(seed)           // Mersenne Twister
difficulty = getDayDifficulty(dayOfWeek)
pool = wordPool[difficulty].filter(recentlyUsed)
todayWord = pool[rng.nextInt(pool.length)]
```

#### 자모 테트리스 자모 출현 빈도

```
한국어 자모 빈도표 기반 가중 랜덤:
  고빈도 자모 (ㅇ,ㅏ,ㄴ,ㅡ,ㄹ,ㅣ): 출현 확률 높음
  저빈도 자모 (ㅋ,ㅌ,ㅍ,ㅉ,ㅃ): 출현 확률 낮음 (방해 자모로 활용)

난이도 곡선:
  경과 시간에 따라 낙하 속도 증가 (0.5초/칸 → 0.15초/칸)
  저빈도 자모 출현률 점진적 증가
```

### 콘텐츠 수량 계획

| 시기 | 단어 DB | 게임 모드 | 테마 | 기타 |
|------|---------|----------|------|------|
| 출시 시 | 50,000+ 단어 | 핵심 3축 + 보너스 3모드 | 기본 4종 + 계절 1종 | 위젯, 리그 시즌 1 |
| 3개월 | 신조어/유행어 500+ 추가 | 유저 출제 퍼즐 런칭 | 한글날 특별 테마 | 연말 리캡 시스템 |
| 6개월 | 사자성어/속담 팩 2,000+ | 팀 대결 모드 | 명절 테마 2종 추가 | 해외 한국어 학습자 모드 |
| 1년 | 총 55,000+ 단어 | 커뮤니티 추천 모드 1종 | 총 12종+ 테마 | 리그 시즌 12회 완료 |

### 시즌/이벤트 설계

**월간 시즌 (한끝차이 리그)**:
- 매월 1일 시작, 말일 종료
- 시즌 성적 기반 등급 산정: 브론즈 → 실버 → 골드 → 플래티넘 → 다이아몬드 → 한글 마스터
- 시즌 종료 시 등급 카드 자동 생성 + 한정 테마/프레임 보상

1. **단어 도감**: 정답으로 등장한 모든 단어를 수집. 뜻/예문/용례 제공. 카테고리별 도감 완성도 표시
2. **테마 컬렉션**: 기본/계절/이벤트/한정 테마 수집. 테마별 미리보기 + 해금 조건 표시
3. **칭호 컬렉션**: 마일스톤/시즌/이벤트 칭호 수집
4. **리캡 아카이브**: 과거 주간/월간/연말 리캡 카드 열람

### 업적/도전과제 시스템

**일일 업적**:

**숨겨진 업적**:

**자모 테트리스 1v1 대결**:

- 방해 시스템: 콤보 달성 시 상대에게 저빈도 자모(ㅋ, ㅌ, ㅍ) 전송

### 애니메이션 원칙

1. **글자 완성 모션**: 초성+중성+종성이 하나의 글자로 합쳐지는 0.5초 연출 — 게임의 시그니처
2. **피드백 뒤집기**: 각 자모 칸이 순차적으로 뒤집히며 색상 표시 (0.3초/칸, 좌→우 순차)
3. **스트릭 불꽃**: 스트릭 카운터에 연속일수에 비례하는 불꽃 이펙트
4. **계절 파티클**: 테마에 따른 배경 파티클 (벚꽃잎, 눈송이, 낙엽 등) — 미세하게
5. **이징(Easing)**: 모든 UI 전환에 ease-out 적용, 자연스럽고 부드러운 느낌
6. **마이크로 인터랙션**: 버튼 탭 시 미세한 스케일 바운스 (0.95→1.0), 성공 시 confetti

## 10. 기술 요구사항

### 엔진/프레임워크 추천

| 옵션 | 추천도 | 근거 |
|------|-------|------|
| **Flutter** | ★★★★★ | 크로스플랫폼 효율, 커스텀 키보드 UI 구현 용이, 애니메이션 성능 우수, 위젯 지원 |
| React Native | ★★★☆☆ | 크로스플랫폼이나 커스텀 애니메이션 한계, 네이티브 모듈 필요할 수 있음 |
| Swift + Kotlin (네이티브) | ★★★★☆ | 키보드 UX 최적화, 위젯 구현 최적. 개발 비용 2배 |

**추천**: Flutter를 1차 선택지로 하되, 키보드 입력 UX의 네이티브 수준 구현이 어려울 경우 Swift/Kotlin 네이티브로 전환

### 핵심 기술 과제

1. **한글 자모 분리/조합 엔진**: 유니코드 기반 초성/중성/종성 분리 및 피드백 생성 로직
2. **시드 기반 PRNG 동기화**: 모든 기기에서 동일 날짜에 동일 퍼즐 보장 (Mersenne Twister)
3. **커스텀 한글 키보드**: OS 키보드를 대체하는 인앱 키보드 (자모 상태 표시 포함)
4. **P2P 실시간 대결**: WebRTC 또는 로컬 Wi-Fi 기반 자모 테트리스 1v1 (서버 불필요)
5. **위젯 개발**: iOS WidgetKit / Android AppWidget 구현 + 딥링크
6. **한글 아트 애니메이션**: 자모 조합 모션 그래픽의 부드러운 렌더링
