# 프로젝트 지침

이 프로젝트는 모바일 게임입니다. `docs/GAME_DESIGN.md`에 상세 기획문서가 있습니다.

## 작업 지침
1. `docs/GAME_DESIGN.md`를 먼저 읽으세요
2. 기획문서의 기술 요구사항 섹션에 명시된 엔진/프레임워크를 사용하세요
3. MVP(최소 실행 가능 제품) 우선으로 코어 게임플레이부터 구현하세요

## 구현 순서
1. 프로젝트 초기화 (엔진 설정, 디렉토리 구조)
2. 코어 메커니즘 (핵심 게임플레이)
3. UI/UX (화면 구성, 온보딩)
4. 콘텐츠 시스템 (프로시저럴 생성)
5. 소셜/공유 기능
6. 수익화 연동

## 중요 원칙
- 질문 없이 자동으로 진행하세요
- 오프라인/서버리스 우선 아키텍처
- 모바일 성능 최적화 (60fps, 저메모리)
