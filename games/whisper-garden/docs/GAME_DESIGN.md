# Whisper Garden (속삭이는 정원) - 심화 기획문서

---

## 1. 게임 비전

### 한줄 소개 (엘리베이터 피치)
> "실시간으로 자라나는 나만의 정원을 가꾸고, 보고, 듣고, 공유하는 감성 힐링 시뮬레이션 — 기다림이 선물이 되는 곳."

### 게임의 핵심 감정 (플레이어가 느껴야 할 것)
- **평온함**: 앱을 여는 순간 일상의 소음이 사라지고, 정원의 자연음이 마음을 감싸는 느낌
- **기대감**: "내 벚나무가 오늘 피었을까?" — 오프라인 성장 결과를 확인하는 설렘
- **성취감**: 30일간 기다린 전설 식물의 개화, 아름다움 점수 만점 달성, 비밀 레시피 발견
- **소속감**: 내 정원을 누군가 방문해 물을 주고 감탄하는 연결의 따뜻함
- **자부심**: "이건 내가 만든 작품이야" — 시각+청각을 동시에 디자인한 감각 정원의 완성

### 디자인 철학 (왜 이 게임이 존재하는가)
Whisper Garden은 "빠르고 자극적인 것"이 지배하는 모바일 게임 시장에서 정반대의 가치를 제안한다. 기다림을 벌칙이 아닌 보상으로, 느림을 지루함이 아닌 풍요로움으로 전환하는 것이 핵심 철학이다. 이 게임은 대중성 경쟁이 아닌, 소수의 충성 유저에게 깊은 경험을 제공하여 높은 LTV(생애 가치)를 만들어내는 "니치 강자" 전략을 취한다. 총점이 아닌 수익성, 다운로드 수가 아닌 리텐션과 ARPU가 이 게임의 진정한 KPI다.

### 타겟 유저 프로필

**1차 타겟 (코어)**
- P2 이서연형 (25~35세 직장 여성): 출퇴근길 힐링, 인스타 감성 공유, 충동 구매형 소액 과금
- P4 정미경형 (35~50세 주부): 자투리 시간 힐링, 카톡 기반 소셜, 수면 루틴으로 매일 밤 접속
- P6 한소율형 (25~40세 아트 감성 유저): 정원 디자이너 모드를 창작 도구로 활용, 포트폴리오 구축, 개발자 응원 과금

**2차 타겟**
- ASMR/수면 콘텐츠 소비자: 릴렉스 타이머를 수면 앱으로 매일 밤 활용
- 수집형 캐주얼 게이머: 도감 완성, 비밀 레시피 발견에 동기 부여

**비타겟 (의도적 배제)**
- P1 김하율형 (10대 남성): "꽃 키우기 = 언니들 게임" 인식. 간접 바이럴(가족 추천)만 기대
- P3 박준혁형 (30대 하드코어 게이머): 시스템 깊이 구조적 한계. 기술 존경 차원 후원만 가능
- P5 최동현형 (20대 경쟁형 남성): 힐링/감성 게임과 180도 반대 성향. 공략 불가능

## 2. 코어 메커니즘 상세

### 핵심 동사 (플레이어가 하는 행동)
1. **가꾸다** — 식물에 물을 주고, 가지를 치고, 벌레를 잡아 성장을 돕는다
2. **꾸미다** — 식물과 장식물을 배치하고, 지형을 편집하여 나만의 정원을 디자인한다
3. **발견하다** — 비밀 조합 레시피를 실험하고, 희귀 식물과 계절 미스터리를 해독한다
4. **감상하다** — ASMR 사운드스케이프를 커스터마이징하고, 정원 산책과 수면 모드를 즐긴다
5. **공유하다** — 포토 모드로 정원을 촬영하고, 정원 일기를 SNS에 나누고, 이웃 정원을 방문한다

### 게임플레이 플로우차트

```
[앱 실행]
    │
    ▼
[오프라인 성장 결과 확인] ──── "벚나무가 만개했어요!"
    │
    ▼
[오늘의 정원 일기 카드 수령] ──── 감성 카드 → 즉시 SNS 공유 가능
    │
    ▼
[데일리 퀘스트 확인] ──── "장미에 물 주기" / "벌 3마리 관찰하기"
    │
    ├──→ [관리 미니게임] ──── 물주기 리듬게임 / 가지치기 퍼즐 / 벌레잡기
    │         │
    │         ▼
    │    [즉시 성장 부스트] ──── 1~4시간 단축
    │
    ├──→ [정원 꾸미기] ──── 식물 심기 / 장식 배치 / 지형 편집
    │         │
    │         ▼
    │    [아름다움 점수 확인] ──── 색상 조화 / 배치 밸런스 / 테마 보너스
    │
    ├──→ [정원 디자이너 모드] ──── 자유 배치 / 조경 도구 / 사운드 가든 디자인
    │         │
    │         ▼
    │    [포트폴리오 저장] ──── 시각+청각 감각 작품 완성
    │
    ├──→ [비밀 레시피 실험] ──── 식물 조합 배치 → 희귀 식물 발견
    │
    ├──→ [소셜 활동] ──── 이웃 정원 방문 / 선물 보내기 / 멘토링
    │
    ├──→ [포토 모드] ──── 촬영 → 주간 콘테스트 참여 → SNS 공유
    │
    └──→ [릴렉스 타이머] ──── ASMR 수면 모드 → 페이드 아웃 → 기상 알람
              │
              ▼
         [앱 종료 — 식물은 계속 성장]
```

### 조작 방식 (터치 제스처별 매핑)

| 제스처 | 일반 모드 | 디자이너 모드 | 포토 모드 |
|--------|-----------|---------------|-----------|
| **탭** | 식물 선택 / 물주기 / 미니게임 입력 | 오브젝트 선택 | 셔터 버튼 |
| **더블 탭** | 식물 상세 정보 | 오브젝트 복제 | 자동 포커스 |
| **롱프레스** | 식물 이동 모드 진입 | 오브젝트 속성 패널 | 타이머 촬영 |
| **드래그** | 카메라 이동 (패닝) | 오브젝트 이동 / 경로 드로잉 | 카메라 이동 |
| **핀치 줌** | 정원 확대/축소 | 개별 식물 크기 조절 | 줌 인/아웃 |
| **두 손가락 회전** | 카메라 회전 | 오브젝트 회전 | 카메라 앵글 |
| **스와이프 업/다운** | 시간대 미리보기 (아침↔밤) | 레이어 전환 (전경/중경/후경) | 필터 전환 |

### 난이도 곡선 설계

이 게임은 전통적 "난이도"가 아닌 "깊이의 점진적 개방"으로 설계한다.

```
Day 1~3:   [입문] 기본 식물 심기, 물주기, 첫 번째 개화 경험
Day 4~7:   [확장] 장식물 배치, 아름다움 점수 시스템 소개, 첫 비밀 레시피 힌트
Day 8~14:  [심화] 계절 시스템 체험, 정원 디자이너 모드 해금, 이웃 방문 시작
Day 15~30: [숙련] 아름다움 점수 최적화, 공간 제한 챌린지, 사운드 가든 디자인
Day 31~60: [마스터] 비밀 레시피 도감 도전, 포토 콘테스트 참여, 멘토링
Day 61+:   [장인] 시즌 이벤트 수집, 포트폴리오 구축, 커뮤니티 리더
```

### 세션 설계

| 구분 | 시간 | 활동 | 타겟 상황 |
|------|------|------|-----------|
| **최소 세션** | 1~2분 | 오프라인 성장 확인 + 정원 일기 수령 + 물주기 | 바쁜 아침, 지하철 탑승 직후 |
| **최적 세션** | 5~10분 | 미니게임 + 꾸미기 + 비밀 레시피 실험 + 포토 촬영 | 점심시간, 퇴근 후 |
| **최대 세션** | 20~40분 | 디자이너 모드 + 사운드 가든 디자인 + 이웃 방문 + 콘테스트 | 주말 오후 |
| **수면 세션** | 15~60분 | 릴렉스 타이머 ASMR 수면 모드 (점진적 페이드 아웃) | 매일 밤 취침 전 |

## 3. 게임 모드별 상세

### 모드 1: 마이 가든 (핵심 모드)
- **목적**: 실시간 성장 기반 정원 가꾸기의 메인 경험
- **규칙**: 실제 시간 경과에 따라 식물 성장 (6시간~30일), 계절/날씨 시스템 연동
- **승리 조건**: 없음 (오픈엔드). 마일스톤: 식물 도감 완성도, 정원 레벨, 아름다움 점수
- **타겟**: P2 (출퇴근 힐링), P4 (일상 루틴), P6 (아트 감성)

### 모드 2: 정원 디자이너 (창작 모드)
- **목적**: 그리드 제약 없는 자유 배치로 인테리어/조경 시뮬레이션 수준의 창작 경험
- **규칙**: 자유 회전, 크기 조절, 레이어 시스템 (전경/중경/후경), 조경 도구 (길/지형/조명)
- **승리 조건**: 없음. 포트폴리오 구축, 디자이너 콘테스트 참여가 동기
- **타겟**: P6 (창작 도구 활용), P2 (인테리어 감성)

### 모드 3: 사운드 가든 디자인 (감각 창작 모드)
- **목적**: 시각적 정원 + 청각적 정원을 동시에 디자인하는 멀티감각 창작
- **규칙**: 배치한 오브젝트에 따라 ASMR 사운드가 바이노럴 3D 오디오로 공간 반응
- **승리 조건**: 없음. "보고 듣는 작품" 완성 자체가 목표
- **타겟**: P6 (예술 창작 도구), P2 (감성 경험)

### 모드 4: 릴렉스 타이머 (수면/명상 모드)
- **목적**: 수면 앱 수준의 ASMR 경험으로 매일 밤 접속 루틴 형성
- **규칙**: 15/30/45/60분 타이머, 점진적 볼륨 감소, 가이드 오디오, 수면 통계
- **승리 조건**: 없음. 수면 품질 개선, 연속 사용일 기록이 동기
- **타겟**: P4 (매일 밤 수면 루틴), P2 (스트레스 해소)

### 모드 5: 아름다움 점수 챌린지 (퍼즐 모드)
- **목적**: 제한된 공간에서 아름다움 점수를 최대화하는 최적화 퍼즐
- **규칙**: 4x4, 6x6 등 제한 타일에서 색상 조화, 배치 밸런스, 다양성, 테마 통일성 극대화
- **승리 조건**: 각 챌린지별 목표 점수 달성 / 만점 도전
- **타겟**: P6 (조경 설계), P2 (꾸미기 퍼즐)

### 모드 6: 주간 포토 콘테스트 (커뮤니티 모드)
- **목적**: 정원 스크린샷을 공유하고 커뮤니티 투표로 순위를 경쟁
- **규칙**: 매주 테마 부여, 포토 모드 촬영 → 코드 인코딩 비동기 참여 → 좋아요 투표
- **승리 조건**: 주간 TOP 20 진입 → 희귀 씨앗/장식 보상
- **타겟**: P2 (인스타 감성), P6 (아트 작품 전시), P4 (맘카페 자랑)

### 모드 간 관계 (계층 구조)

```
[핵심] 마이 가든 ─────── 매일 접속의 기본 루프
  │
  ├── [서브] 정원 디자이너 ─── 마이 가든의 창작 확장
  │     └── [보너스] 사운드 가든 디자인 ─── 디자이너 + ASMR 시너지
  │
  ├── [서브] 아름다움 점수 챌린지 ─── 꾸미기의 퍼즐화
  │
  ├── [서브] 릴렉스 타이머 ─── 매일 밤 수면 루틴 (DAU 유지 핵심)
  │
  └── [서브] 주간 포토 콘테스트 ─── 커뮤니티 + 바이럴 엔진
```

## 4. 콘텐츠 시스템

### 프로시저럴 생성 알고리즘 설계

**식물 변이 시스템**
- 기본 식물 모델: L-System(린덴마이어 시스템) 기반 프로시저럴 생성
- 변이 파라미터: 색상 HSV(3~5변이) x 크기(소/중/대) x 형태(기본/곡선/풍성)
- 총 생성 공간: 100종 x 5색상 x 3크기 x 3형태 = 약 4,500가지 시각적 변이
- 희귀도 결정: 조합 조건 난이도에 따라 일반/희귀/전설/시즌 등급 자동 배정

**사운드 믹싱 알고리즘**
- 정원 상태 파라미터(식물 수, 물 오브젝트 유무, 시간대, 계절)에 따라 실시간 사운드 레이어링
- 바이노럴 3D 오디오: 오브젝트 좌표 기반 HRTF(머리전달함수) 연산으로 공간 음향 배치
- 레이어 합성: 최대 12레이어 동시 믹싱 (환경음 4 + 인터랙션 4 + BGM 2 + 특수 2)

**계절/날씨 자동 순환**
- System.DateTime 기반 4계절 자동 전환 (3~5월 봄, 6~8월 여름, 9~11월 가을, 12~2월 겨울)
- 셰이더 파라미터 보간으로 조명/색감 자연 전환 (새벽/아침/낮/저녁/밤 5단계)
- 날씨: 디바이스 위치 API(선택적) 또는 Perlin Noise 기반 랜덤 날씨 패턴 생성

### 콘텐츠 수량 계획

| 시점 | 식물 | 장식물 | 비밀 레시피 | 테마 가든 | 사운드팩 |
|------|------|--------|-------------|-----------|----------|
| **출시 시** | 100종 (꽃 30 + 나무 15 + 관목 15 + 채소/허브 20 + 판타지 20) | 50종 | 50종 | 5종 (젠/코티지/열대/모던/클래식) | 4종 (계절별) |
| **3개월** | +20종 (시즌 2종x2 + 이벤트 한정 16종) | +15종 | +10종 | +2종 | +2종 (콜라보) |
| **6개월** | +40종 (누적 160) | +30종 | +20종 | +3종 | +4종 |
| **1년** | +80종 (누적 220) | +60종 | +40종 (누적 90) | +5종 (누적 15) | +8종 (연간 구독자 보상 포함) |

### 시즌/이벤트 설계 (서버 없이 가능한 방식)

모든 시즌 이벤트는 디바이스 날짜 기반 자동 활성화로 서버 없이 운영된다.

```
시즌 패스 ≠ 가든 클럽: 별도 상품. 둘 다 구매하면 최고 프리미엄 경험
```

### 타이포그래피 가이드
- **제목/헤더**: 둥근 고딕 계열 (예: Pretendard ExtraBold 또는 나눔스퀘어라운드 Bold) — 부드럽고 친근한 인상
- **본문/설명**: Pretendard Regular — 높은 가독성, 깔끔한 한글 렌더링
- **정원 일기/감성 텍스트**: 나눔 손글씨 또는 커스텀 손글씨 폰트 — 직접 쓴 듯한 따뜻한 느낌
- **숫자/데이터**: Pretendard Medium (등폭) — 점수, 타이머 등 정보 표시
- **크기 기준**: 제목 24pt / 서브타이틀 18pt / 본문 14pt / 캡션 12pt

### 사운드 디자인 방향

**환경음 레이어링 (최대 12레이어 동시)**
- 레이어 1~4: 기본 환경음 (바람/물/새/벌레 — 정원 상태에 따라 동적 변화)
- 레이어 5~8: 인터랙션 사운드 (물주기/흙파기/수확/개화 — 촉각적 ASMR)
- 레이어 9~10: BGM (미니멀 피아노/어쿠스틱 — 시간대별 자동 전환)
- 레이어 11~12: 특수 효과 (계절 이벤트/날씨/희귀 이벤트)

**ASMR 설계 원칙**
- 모든 인터랙션 사운드는 바이노럴 녹음 기반 (이어폰 착용 시 3D 공간감)
- 사운드 라이브러리: 각 식물/장식물별 최소 3종 변이 사운드 (반복감 방지)
- 물 소리 = 7종 변이, 흙 소리 = 5종, 풀잎 소리 = 6종, 바람 = 4종(계절별)

**BGM 설계**
- 아침 (6~10시): 밝고 경쾌한 피아노 아르페지오
- 낮 (10~17시): 따뜻한 어쿠스틱 기타 + 가벼운 스트링
- 저녁 (17~21시): 잔잔한 앰비언트 패드 + 피아노
- 밤 (21~6시): 조용한 피아노 솔로 + 귀뚜라미/풀벌레 (릴렉스 전환)

### 애니메이션 원칙
- **Ease-in-out 기본**: 모든 UI 전환은 부드러운 가감속 (300~500ms)
- **자연적 흔들림**: 식물은 미세한 바람 흔들림 셰이더 (Vertex Animation) 항상 적용
- **개화 연출**: 꽃이 피는 순간 1.5초 슬로모션 + 파티클 이펙트 + 전용 사운드
- **계절 전환**: 3일에 걸쳐 점진적으로 색감/조명이 변화 (급격한 전환 없음)
- **터치 피드백**: 모든 터치에 미세한 스케일 바운스 (0.95→1.0) + 햅틱
- **카메라 이동**: 관성 기반 스무딩 (감속 곡선) — 급격한 멈춤 없음

## 10. 기술 요구사항

### 엔진/프레임워크 추천

| 영역 | 기술 | 선정 근거 |
|------|------|-----------|
| 게임 엔진 | **Unity 2022 LTS + URP** | 2D/3D 하이브리드 렌더링, 크로스플랫폼, 커스텀 셰이더 용이 |
| 렌더링 | **Universal Render Pipeline + 커스텀 셰이더** | 시간대/날씨별 조명 변화, 수채화 텍스처 효과, 식물 흔들림 |
| 식물 생성 | **L-System 기반 프로시저럴** | 적은 에셋으로 수백 종의 식물 형태 + 색상 변이 자동 생성 |
| 사운드 | **FMOD Studio** | 다중 레이어 앰비언트 믹싱, 바이노럴 3D 오디오, 파라미터 기반 동적 사운드 |
| 저장 | **SQLite (로컬)** | 식물 상태, 정원 레이아웃, 도감 등 구조화 데이터 |
| 시간 관리 | **System.DateTime + AES 암호화** | 오프라인 성장 계산, 시간 조작 방지 |
| 포토 모드 | **RenderTexture + Post-Processing Stack** | 필터/프레임 적용 고해상도 캡처 (최대 4K) |
| 광고 SDK | **Google AdMob** | 리워드 광고 전용, 안정적 Fill Rate |
| IAP | **Unity IAP + RevenueCat** | iOS/Android 통합 결제, 구독 관리 |
| 분석 | **Firebase Analytics** | 오프라인 이벤트 캐싱, 구독 전환 퍼널 분석 |

### 핵심 기술 과제

1. **바이노럴 3D 오디오 실시간 연산**: 오브젝트 좌표 기반 HRTF 연산을 모바일에서 실시간 처리. FMOD의 공간 오디오 플러그인 활용 + 동시 사운드 소스 제한(최대 8개 동시 3D 소스)으로 해결
2. **L-System 프로시저럴 식물의 시각적 품질**: 알고리즘 생성 식물이 수작업 에셋 수준의 아름다움을 가져야 함. L-System 기본 형태 + 수채화 텍스처 랜덤 매핑 + 후처리 셰이더로 해결
3. **오프라인 성장 시간 조작 방지**: System.DateTime 기반이므로 디바이스 시간 변경 공격 가능. 마지막 접속 시간 암호화 저장 + 비정상 시간 점프 감지 + 서버 없이 로컬 검증 로직
4. **수백 종 식물의 메모리 관리**: LOD(Level of Detail) 시스템 + 화면 밖 식물 메시 언로드 + 텍스처 아틀라스 최적화

### 오프라인/서버리스 아키텍처

```
[디바이스 로컬]
    │
    ├── SQLite DB ──── 식물 상태, 정원 레이아웃, 도감, 업적, 설정
    │
    ├── 성장 엔진 ──── System.DateTime 기반 오프라인 성장 계산
    │                  (앱 실행 시 경과 시간 계산 → 성장 적용)
    │
    ├── 프로시저럴 엔진 ── L-System 식물 생성 + Perlin Noise 날씨
    │
    ├── 사운드 엔진 ──── FMOD 로컬 렌더링 + 바이노럴 3D
    │
    ├── 공유 시스템 ──── OS 네이티브 Share Sheet (이미지/영상/딥링크)

    ├── 이웃 방문 ──── 정원 코드 = 정원 데이터의 압축 시리얼라이즈
    │                  (8자리 코드 or QR → 디코드 → 읽기 전용 렌더링)
    │
    └── 클라우드 백업 ── iCloud / Google Drive (선택적, 유저 데이터만)
```

- 서버 완전 불필요: 모든 연산 로컬
- 소셜 기능: 코드 기반 비동기 (정원 코드 = 직렬화 데이터)

- 시즌 이벤트: 디바이스 날짜 기반 자동 활성화

## 12. 리스크 & 대응

| 리스크 | 영향도 | 확률 | 대응 방안 |
|--------|--------|------|-----------|
| **좁은 타겟으로 인한 다운로드 규모 제한** | 높음 | 높음 | 총점/다운로드가 아닌 ARPU/LTV를 핵심 KPI로 설정. 코어 유저 3천 명이 월 ₩10,000 과금하면 월 3천만 원 — 소규모 팀 기준 충분한 수익. 규모보다 수익성에 집중 |
| **수면 앱/힐링 앱과의 포지셔닝 혼동** | 중간 | 중간 | "정원 가꾸기 게임"이 핵심 정체성. 수면 모드는 부가 가치이지 메인이 아님을 앱스토어 설명/스크린샷에서 명확히 커뮤니케이션 |
| **오프라인 성장 시간 조작 (치트)** | 중간 | 높음 | 마지막 접속 시간 AES 암호화 + 비정상 시간 점프 감지 (48시간 초과 성장 시 보정 적용). PvP가 없으므로 치트의 실질적 피해 제한적 |
| **L-System 식물 시각적 품질 미달** | 높음 | 중간 | 핵심 50종은 수작업 에셋으로 제작, 변이종만 프로시저럴. 소프트론칭에서 유저 피드백 수집 후 품질 기준 미달 식물 교체 |
| **바이노럴 3D 오디오 저사양 기기 성능 이슈** | 중간 | 중간 | 저사양 모드: 3D 오디오 → 스테레오 폴백. 동시 사운드 소스 8개 → 4개로 축소. 사운드 가든 디자인 기능은 권장 사양 이상에서만 활성화 |
| **가든 클럽 구독 피로도** | 중간 | 낮음 | 비구독자도 핵심 기능 100% 이용 가능하게 유지. 구독은 "프리미엄 경험 강화"이지 "필수"가 아님. 구독 취소 유저에게 3일 후 "다시 오세요" 할인 오퍼 (₩2,900/월 첫 달) |
| **계절 이벤트 콘텐츠 소진 (2년차 반복)** | 중간 | 중간 | 연도별 시즌 한정 식물을 다르게 설계 (2026 봄 vs 2027 봄 한정 식물 다름). 프로시저럴 변이로 매년 새로운 느낌 제공 |
| **힐링 게임 시장 포화** | 낮음 | 중간 | "정원 + ASMR + 수면 + 사운드 디자인"의 멀티감각 경험은 단순 힐링 게임과 차별화. 진입 장벽이 높아 쉽게 복제되지 않음 |
